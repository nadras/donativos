🎉 ¡Casi listo!
Sigue estos últimos pasos para tener el sistema funcionando:
1
Publica el Apps Script como Web App En Google Apps Script → Implementar → Nueva implementación → Tipo: App web → Ejecutar como: Yo → Acceso: Cualquier persona. Copia la URL y pégala en el Paso 5.
2
Verifica los emails de los líderes En Apps Script, selecciona la función grantEmailPermission y ejecútala una vez para aceptar los permisos de Gmail.
3
Conecta las URLs finales En el Paso 5, pega la URL del Web App y la URL pública de upload.html. Copia los bloques generados en Code.gs, form.html y upload.html.
4
Sube form.html a un hosting o usa GitHub Pages Puedes usar GitHub Pages (gratis) o cualquier hosting. Ese link será la URL de tu QR.
5
Genera el código QR Usa qr.io o qrcode-monkey.com con la URL de tu form.html. Imprime y coloca en el centro.
6
Haz una prueba completa Llena el formulario, guarda el enlace en WhatsApp, y sube una foto de prueba. Verifica que el Sheets y el Drive tengan los datos correctos.
📁 ¿Dónde están tus archivos?
Los archivos que debes alojar son: form.html y upload.html. El Code.gs va en Google Apps Script — no se aloja como archivo.
